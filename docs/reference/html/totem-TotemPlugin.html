Description

libpeas is used as a general-purpose architecture for adding plugins to Xplayer, with derived support for different programming languages.

The functions in xplayer-dirs.h are used to allow plugins to find and load files installed alongside the plugins, such as UI files.

xplayer_plugin_find_file ()

char *              xplayer_plugin_find_file              (const char *plugin_name,
                                                         const char *file);

Finds the specified file by looking in the plugin paths listed by xplayer_get_plugin_paths() and then in the system Xplayer data directory.

This should be used by plugins to find plugin-specific resource files.

plugin_name :

 the plugin name

file :

 the file to find

Returns :

 a newly-allocated absolute path for the file, or NULL

xplayer_plugin_load_interface ()

GtkBuilder *        xplayer_plugin_load_interface         (const char *plugin_name,
                                                         const char *name,
                                                         gboolean fatal,
                                                         GtkWindow *parent,
                                                         gpointer user_data);

Loads an interface file (GtkBuilder UI file) for a plugin, given its filename and assuming it's installed in the plugin's data directory.

This should be used instead of attempting to load interfaces manually in plugins.

plugin_name :

 the plugin name

name :

 interface filename

fatal :

 TRUE if it's a fatal error if the interface can't be loaded

parent :

 the interface's parent GtkWindow. [allow-none]

user_data :

 a pointer to be passed to each signal handler in the interface when they're called. [allow-none]

Returns :

 the GtkBuilder instance for the interface. [transfer full]

xplayer_get_plugin_paths ()

char **             xplayer_get_plugin_paths              (void);

Return a NULL-terminated array of paths to directories which can contain Xplayer plugins. This respects the GSettings disable_user_plugins setting.

Returns :

 a NULL-terminated array of paths to plugin directories. [transfer full]
